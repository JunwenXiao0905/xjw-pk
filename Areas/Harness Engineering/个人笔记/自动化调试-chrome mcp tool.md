● WSL2 MCP 连接 Windows Chrome

  ---
  Windows 端

  # 启动 Chrome（带调试端口）
  & "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

  ---
  WSL2 端

  # 获取 Windows IP
  WIN_IP=$(cat /etc/resolv.conf | grep nameserver | awk '{print $2}')

  # 配置 MCP
  claude mcp remove chrome-devtools
  claude mcp add chrome-devtools --scope user -- npx chrome-devtools-mcp@latest --browser-url=http://${WIN_IP}:9222

  ---
  验证
  claude mcp list  # 应显示 ✓ Connected

  ---
  注意
  - Windows 防火墙可能需开放 9222 端口
  - 每次使用前需先启动 Windows Chrome
