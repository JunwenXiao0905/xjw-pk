## `obsidian-skills` 子能力

| skill | 作用 | 适合场景 |
| ---- | ---- | ---- |
| `obsidian-markdown` | 规范生成 Obsidian Flavored Markdown | 笔记、知识库文档、内部链接、callout、frontmatter |
| `obsidian-bases` | 创建和维护 Obsidian Bases | 项目清单、开源项目库、赛道观察数据库 |
| `json-canvas` | 创建和编辑 Obsidian `.canvas` | 知识地图、项目图谱、方向地图 |
| `obsidian-cli` | 通过 CLI 操作 Obsidian vault | 搜索、读写笔记、追加 daily note、修改 properties |
| `defuddle` | 抽取网页正文并转成干净 Markdown | 收集网页文章、项目介绍、外部资料 |
